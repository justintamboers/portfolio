Reflectie

Welke skills heb ik geleerd in WPL2?

Ik heb vue3, pinia, web API en hoe ik een database kan schrijven bijgeleerd en ook hoe ik deze nieuwe leerstof kan gebruiken voor een website te kunnen maken.

De softskills die ik heb bijgeleerd in WPL2 is vooral communiceren.

Omdat op het begin communiceerde ik wel maar niet altijd duidelijk genoeg of vaak genoeg.

Een voorbeeld van niet duidelijk genoeg is Bv: toen ik voor de groep iets aan het testen was en wat bugfixes doorgaf, begrepen ze wat ik bedoelde niet echt fatsoenlijk dus ik heb geleerd iets duidelijker te communiceren.

Hoe ging het proces in WPL2?

Het proces ging wel goed maar het was niet altijd makkelijk.

In het begin had ik veel moeite vue en web API maar nu gaat het beter met deze stukken.

Maar uiteindelijk vind ik dat ik veel gegroeid ben met coderen in het project van WPL2.
